    Recognizing hand gestures using the Kinect

    Project maintained by arunganesan Hosted on GitHub Pages — Theme by mattgraham

    Recent times have witnessed a large explosion in data. Data visualization has emerged as a powerful technique for data analysis by leveraging human cognition and layout algorithms for quickly seeing patterns. Despite the growth in data visualizations, tools for interacting with the data have largely remained the keyboard and mouse. We hypothesize that by developing accessible and aordable data interaction systems, we can gain more from data visualization analysis. In this project we focus on the gesture recognition aspect of this system.
